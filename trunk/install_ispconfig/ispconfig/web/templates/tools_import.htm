Bookmarks importieren:
Datei:
Den Button "Bookmarks Importieren" bitte nur einmal drücken! 
Das importieren kann 2 - 3 Minuten dauern, je nach Anzahl der Bookmarks.
Internet Explorer
  1. Wählen Sie den Menüpunkt Datei > Importieren / Exportieren in Ihrem Internet Explorer.
  2. Klicken Sie auf weiter und wählen "Favoriten Exportieren"
  3. Treffen Sie die Auswahl der zu exportierenden Einträge
  4. Wählen Sie "Einträge in Datei exportieren" und geben als Datei
    z.B. "C:\bookmark.htm" an. Klicken Sie nun auf fertigstellen.
  5. Klicken Sie nun oben auf dieser Seite auf den "Durchsuchen..."- ("Browse..."-) Button, wählen die zuvor gespeicherte 
    Bookmarkdatei aus und klicken auf Bookmarks Importieren.

    Hinweis: Das Exportieren von Bookmarks aus älteren Versionen des  Internet Explorer ( 4.0 oder älter ) ist nur über ein Bookmark export und kovertierprogramm wie z.B. die Shareware 
    Bookmark Converter 2.x     möglich. 
    Vorgehensweise: konvertieren Sie Ihre Bookmarks in das Netscape Bookmarkformat und importieren Sie diese wie oben unter 5. beschrieben.
Netscape Navigator
  1. Drücken Sie "Strg-B" um Ihre Bookmarks aufzurufen.
  2. Wählen Sie nun den Menüpunkt "File" ("Datei") und darunter den Punkt "Save As ..." ("Sichern unter ...") aus.
  3. Geben Sie als Datei z.B. "C:\bookmark.htm" an.
  4. Klicken Sie nun oben auf dieser Seite auf den "Durchsuchen..."- ("Browse..."-) Button, wählen die zuvor gespeicherte 
    Bookmarkdatei aus und klicken auf Bookmarks Importieren.
